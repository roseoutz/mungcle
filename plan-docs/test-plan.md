## Affected Pages/Routes
- POST /api/auth/kakao — 카카오 OAuth 로그인
- POST /api/auth/email/* — 이메일 회원가입/로그인
- POST/GET/PATCH /api/dogs — 개 프로필 CRUD + 사진 업로드
- POST /api/walks/start|stop — 산책 상태 토글
- GET /api/walks/nearby — 주변 산책자 탐색
- POST /api/greetings — 인사하기 + 응답 + 메시지
- POST /api/reports, /api/blocks — 신고/차단

## Key Interactions to Verify
- 산책 시작 → 주변 탐색에서 나타남 → 인사하기 → 상호 인사 → 간단 메시지 교환
- 차단한 유저가 nearby 결과에서 제외되는지
- GPS 거부 시 수동 동네 선택 fallback
- 60분 후 산책 자동 종료

## Edge Cases
- 200m grid snap 경계값 (적도, 음수 좌표, 0/0)
- 동시에 서로 인사 (race condition → unique constraint)
- 상대가 산책 종료 후 인사 도달
- 이미 인사한 상대에게 중복 인사
- 상호 인사 전 메시지 시도 → 차단
- 30분 위치 공유 만료 후 메시지 → 차단
- 개 등록 없이 산책 시도 → 가드

## Critical Paths
- E2E: 회원가입 → 개 등록 → 산책 시작 → 인사 → 응답 → 메시지 (핵심 가치 루프)
- E2E: 차단 → nearby 제외 확인
- Unit: grid snap 함수 (모든 경계값)
- Unit: 인사 상태 전이 (pending → accepted/expired)
