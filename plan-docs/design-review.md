## Color Usage Rule (중요)

**"유틸리티 vs 감정" 분리:** 같은 앱 안에서 두 색상 역할을 엄격히 구분.

- **Emerald #10B981** = 유틸리티 (산책 토글, 설정 버튼, 선택 상태, 탭바 선택, 프로필 수정, 성향 긍정 태그, 예방접종 체크)
- **Coral #FB7185** = 감정 (인사하기 CTA, 하트 아이콘, 상호 인사 완료 축하, 메시지 보내기, 사회성 하트, 매칭 강조)
- **Amber #F59E0B** = 정보 상태 (응답 대기 타이머, "호기심" 태그, 캐주얼 caution)
- **Red #EF4444** = 오직 파괴적 액션 (삭제, 만료 임박 경고). 인사 거절엔 Red 금지 — slate-100 fill + slate-700 text.

이 구분이 깨지면 앱 인지 부하가 올라감. 절대 유지.

# Revision 3 — Mono Mint v6 (2026-04-09 최종)

**현재 정식 디자인 시스템.** 이전 섹션들은 모두 참조용 이력.

## 철학: One Hue, Many Voices

Emerald + Coral (v4) 같은 dual-color contrast 시스템은 2024 스타일. 색상끼리 싸우면서 "이어지는 느낌"을 방해. 2026 트렌디한 앱들(Linear, Toss, Arc, Notion, Superhuman)은 **단일 hue family로 모든 것을 표현**. Monochromatic = cognitive ease = premium.

v5 Mono Teal이 cohesion은 확보했지만 너무 차가웠음. v6에서 teal → emerald family로 **warm shift** (약 8° hue 이동). 여전히 green family이지만 더 alive, welcoming, 브랜드의 "walk/nature" 코딩 유지.

## Monochromatic Law (깨지면 안 되는 원칙)

**모든 action, emotion, heart, badge, CTA, status, accent는 emerald family 내에서만 해결.** 차이는 opacity와 lightness로. Hue 추가 금지.

- **하트 = emerald** (pink/coral 금지). "social bond" 개념으로 이해.
- **인사하기 버튼 = emerald**. 감정 액션도 emerald.
- **축하 모달 = emerald deep + emerald soft glow**. 다중 컬러 confetti 금지.
- **사회성 dot = emerald**. 채움/비움으로 구분.
- **성향 칩 = emerald variations** (emerald-500 fill vs emerald-100 bg + emerald-700 text로 톤 차등).
- **탭바 active = emerald-600**.
- **Warning amber / Error red**는 functional 예외로만 (타이머 만료, 삭제 확정).

## Typography

- **Headlines**: Plus Jakarta Sans 700, tight tracking `-0.02em`
- **Body**: Plus Jakarta Sans 500
- **Labels/micro**: Inter 600, uppercase tracking `+0.02em`
- **한글 구현**: Pretendard로 교체 필수 (Stitch는 지원 안 함)
- **Scale**: display-xl 40px / display 32px / headline 24px / title 18px / body 14px / caption 12px / micro 11px (큰 jump로 editorial 느낌)

## Shape & Elevation

- Cards: `16px` radius (전 버전 12px → 16px로 softer)
- Chips/inputs: `8px` radius 또는 `full pill`
- Buttons: `full pill` (9999px)
- Avatars: `full circle`
- 카드 분리: tonal layering (white card on emerald-50 section bg). Hard border 금지
- Floating CTA: soft emerald glow `rgba(16, 185, 129, 0.15)` 20px blur
- Modal: backdrop blur 16px + neutral-950 45% overlay
