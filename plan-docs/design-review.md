## Navigation Structure

Bottom Tab Bar: 홈(탐색) | 내 개 | 알림 | 설정

### Screen Map
- **온보딩:** 로그인 → 개 등록 → 동네 설정 → 홈
- **홈:** 산책 토글 (상단) → 주변 개 카드 리스트 → 빈 화면: 시간대 패턴
- **내 개:** 등록된 개 목록 + 추가 버튼
- **알림:** 인사 수신 → 응답 → 메시지/위치 공유
- **설정:** 계정, 차단 관리, 동네 설정

### Information Hierarchy (홈 화면)
1. 산책 상태 토글 + "현재 산책 중 🟢" 배너
2. 주변 개 카드 (사진, 이름, 견종, 성향칩, 사회성 색상, 거리)
3. "인사하기" CTA (카드 안 오른쪽 하단)

## Design Decisions

1. **개 사진 없을 때:** 견종별 실루엣 아바타
2. **성향 태그:** 색상 칩 (녹색 "활발", 파란 "조용", 빨간 "주의")
3. **사회성 점수:** 색상 코딩 (green/yellow/red) 카드 테두리 또는 배지
4. **"인사하기" 버튼:** 카드 안 오른쪽 하단 CTA
5. **알림 배지:** 탭바 알림 탭에 빨간 도트
6. **산책 중 상태:** 헤더 아래 작은 배너 "현재 산책 중 🟢"

## User Journey Emotional Arc

1. 설치 → 기대 + 불안 → 따뜻한 온보딩
2. 개 등록 → 자부심 → 카드 미리보기
3. 홈 도착 → 호기심 → 주변 개 카드 or 시간대 패턴
4. 산책 시작 → 설렘 → "같이 걸어도 좋아요" 배지
5. 인사하기 → 긴장 → "가볍게 인사해보세요" 문구
6. 상호 인사 → 기쁨 → 축하 UI + 메시지 안내
7. 만남 → 보람 → (앱 밖)
